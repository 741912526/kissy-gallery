API

方法

KISSY.gallery.Pagenation ( cfg )

配置项

  • {String|HTMLElement|KISSY.Node} [ container ] 必设. 分页的 DOM 容器
  • {String} [ cls = 'pg-' ] 样式前缀
  • {Function} [ callback ] 点击分页项时, 调用的函数, 其带三个参数: 
        *  - idx: 新页号
    *  - pg obj: 当前分页对象
    *  - ready: fn 供外部调用者, 当切换好页时, 更新分页HTML等后续操作
    * note: 当 callback 返回 false 时, 停止事件冒泡
  • {Number} [ current_page = 0 ] 当前页序号, 从0开始计
  • {Number} [ max_display_page_count = 5] 显示多少页. 务必取奇数, 当为0时, 表示只显示上一页/下一页
  • {Number} [ total_page ] 总共多少页, 当不设时, 不能计算出数字
  • {Boolean} [ has_next = true ] 是否有下一页, 供那些不明确页数情况下使用
  • {Number} [ first_page = 0 ] 首页序号从哪边开始计
  • {String} [ next_text = '下一页' ] 下一页文案
  • {String} [ prev_text = '上一页' ] 上一页文案
  • {Boolean} [ ellipse_text = true ] 是否出现省略符号
  • {Boolean} [ load_first_page = true ] 是否初始加载第一页

方法

  • page(idx): 跳转到指定页.
