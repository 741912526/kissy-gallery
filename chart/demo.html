当前位置： KISSY Gallery Chart(dev) 
				
作者：qipbbn@gmail.com(文龙)
功能：图表工具
描述：基于HTML5 Canvas 的图表引擎
源码：KISSY Chart

功能点

  • 目前支持条形图，折线图
  • 通过 flashCanvas 添加对IE的支持
  • 自动颜色配置，支持颜色自定义
  • 数字显示，格式化
  • 自定义动画效果
  • Y坐标自适应
  • 多种JSON数据定义格式

Demo

条形图（Bar Chart）

{
    type : "bar",
    elements : [
        {
            name : "接待人数",
            datas : [8,29,15,31,5,11],
            label : "{name} : {data}人"
        },
        {
            name : "回复人数",
            datas : [4,29,14,25,6,8],
            label : "{name} : {data}人"
        }
    ],
    axis : {
        x : {
            name : "时间",
            labels : ["2011-1-10","2011-1-11","2011-1-12","2011-1-13","2011-1-14","2011-1-15"]
        },
        y : {
            name : "人数"
        }
    },
    config : {
        left:20
    }
}

折线图(Line Chart)

{
    type : "line",
    elements : [
        {
            name : "接待人数",
            datas : [900,700,800,400,200,500],
            label : "{name} : {data}人"
        },
        {
            name : "回复人数",
            datas : [500,350,156,257,200,321],
            label : "{name} : {data}人"
        },
        {
            name : "满意人数",
            datas : [400,300,null,154,200,230],
            label : "{name} : {data}人"
        }
    ],
    axis : {
        x : {
            name : "时间",
            labels : ["2011-1-10","2011-1-11","2011-1-12","2011-1-13","2011-1-14","2011-1-15"]
        },
        y : {
            name : "人数"
        }
    },
    config : {
        drawbg : 2,
        left : 30
    }
}

饼图(Pie Chart)

{
    type : "pie",
    element : {
        names : ["Internet Explorer","Firefox","Chrome","Safiri","Other"],
        datas : [0.55, 0.216, .119, .0715, .033, .0435],
        label : "{name} : {data}人"
    }
}

API

方法

KISSY.Gallery.Chart(canvas, data)

参数

  • {Sting|HTMLElement} canvas 图表的容器canvas元素
  • {Object} [param图表数据]

返回

null
